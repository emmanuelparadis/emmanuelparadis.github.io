pegas: Population and Evolutionary Genetics Analysis System

pegas is an R package for the analysis of population genetic data.

pegas requires two other R packages: ape (depends) and adegenet (suggests).

pegas is distributed on CRAN and on GitHub. The version on CRAN is 1.3.

The citation for pegas is:
Paradis E. 2010. pegas: an R package for population genetics with an integrated–modular approach.
Bioinformatics 26: 419–420. doi:10.1093/bioinformatics/btp696.
